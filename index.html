📄 Пользовательское соглашение

1. Общие положения

Настоящее соглашение регулирует использование цифрового сервиса и порядок предоставления платной подписки. Используя сервис, пользователь подтверждает согласие с данным соглашением.

2. Описание сервиса

Сервис предоставляет доступ к цифровому онлайн-сервису по подписке. Функциональность сервиса зависит от выбранного тарифа и может изменяться в процессе развития продукта.

3. Условия использования

  • не использовать сервис в незаконных целях
  • не пытаться нарушить работу сервиса
  • не передавать доступ третьим лицам
  • не предпринимать действий, нарушающих стабильность системы

4. Подписка и оплата

Доступ к сервису предоставляется на платной основе. Подписка активируется после подтверждения оплаты платёжной системой.

5. Возвраты

Возврат средств возможен только в случае технической невозможности предоставления доступа.

6. Ответственность

Сервис предоставляется «как есть». Администрация не несёт ответственности за перебои и действия третьих лиц.

7. Изменение условий

Администрация оставляет за собой право изменять условия без уведомления.

🔐 Политика конфиденциальности

1. Общий принцип

Сервис уважает приватность пользователей и минимизирует сбор данных.

2. Политика отсутствия логов

  • история использования сервиса
  • содержимое трафика
  • IP-адреса
  • DNS-запросы
  • временные метки активности

3. Обрабатываемые данные

  • ID пользователя
  • статус подписки
  • технические данные
  • нагрузка системы (агрегировано)

4. Платежи

Платежи обрабатываются через стороннего провайдера. Данные карт не хранятся.

Предоставление доступа к цифровому сервису по подписке.
